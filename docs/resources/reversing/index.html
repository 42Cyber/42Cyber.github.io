El taller, de una hora aproximada de duración, consistió en un temario básico de la ingeniería inversa y el análisis de binarios.

Se trataron las siguientes herramientas:

  • Binutils
  • Strace, Ltrace
  • GDB
  • Binary Ninja
  • Ghidra
